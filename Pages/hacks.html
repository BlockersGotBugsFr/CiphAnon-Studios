CSS Tricks

  • Clean layouts with `:not()`: Simplify selectors and spacing issues. For example, add bottom margin to all elements inside a container *except* the last one.
    Learn more about the `:not()` selector
  • Consistent image sizes: Use the `aspect-ratio` property to automatically size images to a uniform aspect ratio.
    See MDN on `aspect-ratio`
  • Easy theming with CSS variables: Define custom properties for colors and fonts to make site-wide changes from a single location.
    Get the MDN guide on CSS custom properties

JavaScript Tricks

  • Efficient DOM selection: While frameworks offer powerful selection methods, `document.getElementById()` remains the fastest for accessing a single element.
    MDN reference for `getElementById()`
  • Arguments to array: Quickly convert array-like objects into a true array using a spread operator (`...`).
    MDN on the spread syntax
  • Dynamic navigation: Change the URL to another page with a single line of JavaScript.
    Quora: Use `window.location.href` to redirect users
