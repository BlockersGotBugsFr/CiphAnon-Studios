HTML Elements Reference

This page lists all standard HTML5 elements with definitions.

<!DOCTYPE html>
Declares the document type (HTML5).
<html>
The root element that wraps the entire HTML document.
<head>
Container for metadata (not displayed on the page).
<title>
Sets the title of the page (shown in browser tab).
<base>
Sets the base URL for all relative links.
<link>
Links external resources (like CSS, icons).
<meta>
Provides metadata (charset, viewport, description, etc).
<style>
Embeds internal CSS styles.
<body>
Contains all the content visible to users.
<header>
Introductory content or navigation for a page/section.
<footer>
Footer content (credits, copyright, links).
<nav>
Navigation links.
<section>
Thematic grouping of content.
<article>
Self-contained piece of content (e.g., blog post).
<aside>
Side content (like sidebars, callouts).
<h1> â€¦ <h6>
Headings, from most important (h1) to least (h6).
<hgroup>
Groups multiple headings together.
<address>
Contact information.
<p>
Paragraph.
<hr>
Thematic break (horizontal rule).
<pre>
Preformatted text (keeps spaces, tabs, line breaks).
<blockquote>
Quoted block of text.
<ol>
Ordered list (numbered).
<ul>
Unordered list (bulleted).
<li>
List item.
<dl>
Description list.
<dt>
Term/name in a description list.
<dd>
Definition/description in a description list.
<figure>
Content with optional caption (image, diagram).
<figcaption>
Caption for a <figure>.
<main>
Main content of a document.
<div>
Generic container for grouping.
<a>
Hyperlink.
<em>
Emphasized text (usually italic).
<strong>
Important text (usually bold).
<small>
Smaller text (fine print).
<s>
Strikethrough (no longer relevant).
<cite>
Title of a creative work.
<q>
Inline quotation.
<dfn>
Definition of a term.
<abbr>
Abbreviation or acronym.
<ruby>
Ruby annotation (used in East Asian typography).
<rb>
Ruby base text.
<rt>
Ruby text (annotation).
<rtc>
Ruby text container.
<rp>
Fallback parentheses for ruby text.
<data>
Associates content with machine-readable data.
<time>
Represents a specific time/date.
<code>
Inline code snippet.
<var>
Variable in math or programming.
<samp>
Sample output from a program/system.
<kbd>
Keyboard input.
<sub>
Subscript text.
<sup>
Superscript text.
<i>
Text offset from normal (italic, technical terms).
<b>
Bold text (stylistic, not importance).
<u>
Underlined text.
<mark>
Highlighted/marked text.
<bdi>
Isolates text for bidirectional formatting.
<bdo>
Overrides text direction.
<span>
Generic inline container.
<br>
Line break.
<wbr>
Optional word break.
<ins>
Inserted text (often shown as underlined).
<del>
Deleted text (often shown as strikethrough).
<picture>
Responsive images container.
<source>
Defines multiple media sources.
<img>
Image.
<iframe>
Inline frame (embed another HTML page).
<embed>
Embeds external content (like plugins).
<object>
Embeds external resources (media, apps).
<param>
Defines parameters for <object>.
<video>
Video player.
<audio>
Audio player.
<track>
Text tracks (subtitles, captions) for <video>/<audio>.
<map>
Defines an image map.
<area>
Clickable area in an image map.
<canvas>
Drawing surface (JavaScript API).
<svg>
Vector graphics.
<math>
Mathematical notation (MathML).
<table>
Table container.
<caption>
Table caption.
<colgroup>
Group of columns.
<col>
Defines properties for a column.
<thead>
Table header section.
<tbody>
Table body section.
<tfoot>
Table footer section.
<tr>
Table row.
<td>
Table cell (data).
<th>
Table header cell.
<form>
Input form.
<label>
Label for a form control.
<input>
Input field.
<button>
Button.
<select>
Dropdown list.
<datalist>
Predefined options for an <input>.
<optgroup>
Group of <option> elements.
<option>
Option in a list.
<textarea>
Multiline text input.
<output>
Output value.
<progress>
Progress bar.
<meter>
Scalar measurement within a range.
<fieldset>
Groups related form controls.
<legend>
Caption for a <fieldset>.
<details>
Expandable/collapsible content.
<summary>
Summary/label for <details>.
<dialog>
Dialog box/modal.
<slot>
Placeholder for content in a web component.
<template>
Template for reusable content.
